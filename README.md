# WatchPartyTimer 🕒

**同時視聴をもっと快適に。経過時間と「残り時間」が同時にわかる配信者向けタイマー。**

---

## 💡 概要 (Overview)
このツールは、映画やアニメの同時視聴配信を行うストリーマー向けに開発された、ブラウザベースのタイマーです。
「今、開始から何分？」だけでなく、**「あと何分で終わるのか」**を一目で把握できるのが特徴です。

## ✨ 特徴 (Features)
- **デュアル表示**: 「経過時間」と「残り時間」を同時にカウント。
- **OBS最適化**: クロマキー合成しやすい背景色、視認性の高いフォントを採用。
- **軽量・シンプル**: HTMLファイル1つで動作。複雑な設定は不要です。
- **レスポンシブ**: PC、タブレット、スマホのブラウザで動作。

## 🛠️ 技術スタック (Tech Stack)
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- Developed on Galaxy with [Spck Code Editor]

## 🚀 使い方 (How to Use)
1. 本リポジトリの `index.html` をダウンロードします。
2. ブラウザでファイルを開きます。
3. 動画の総時間を入力し「START」をクリック。
4. OBS等の配信ソフトに「ブラウザソース」または「ウィンドウキャプチャ」で取り込んでください。
